# KinKeep Discord Operating System

Zurg note: this file is the server/channel structure to create manually unless Discord admin API tools are available in the environment.

## Category: KinKeep
- `#kinkeep` — main daily command center
- `#kinkeep-build` — app build updates, live links, bugs, deploy notes
- `#kinkeep-design` — logo, UI inspiration, App Store screenshots, brand system
- `#kinkeep-security` — PIN editing, encryption, legacy contacts, abuse prevention, privacy
- `#kinkeep-ideas` — feature backlog, wild ideas, user stories, future versions
- `#kinkeep-b2c-appstore` — consumer App Store strategy, pricing, ASO, screenshots, paywall
- `#kinkeep-b2b-hospitals` — hospital/hospice/palliative care strategy and sales material
- `#kinkeep-reports` — finalized reports, pitch docs, investor/admin summaries

## Suggested channel topics
**#kinkeep-build:** Live app: https://zurg888.github.io/kinkeep-app/ — prototype repo: https://github.com/Zurg888/kinkeep-app

**#kinkeep-design:** Warm Apple-like UI: ivory, parchment, mauve ribbon, dusty rose, gold glow, navy privacy accents. Avoid wedding/memorial-only vibes.

**#kinkeep-security:** Private by default. PIN/Face ID for edits. Edit history. Export anytime. No ads. No AI training on private memories.

**#kinkeep-b2b-hospitals:** B2B strategy for hospitals, hospice, palliative care, oncology, patient experience, chaplaincy, and bereavement programs.

**#kinkeep-b2c-appstore:** Consumer strategy for App Store: family memories, legacy letters, voice notes, subscriptions, retention, ASO.
